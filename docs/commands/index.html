Kick:

Kicks the user from the server. Requires kick permission
Args: 2 or more - user, reason

Ban:

Bans the user from the server. Requires ban permission
Args: 2 or more - user, reason

Ping:

Pings the bot and measures latency in ms
Args: 0

Del:

Deletes an amount of past messages, from all users. Excludes the command
Args: 1

Idea:

Generates random word(s). Can be told how many to generate
Args: 1

Prefix:

Changes the bot prefix for your server. Requires administrator permission.
Args: 1

Info:

Displays bot and server info
Args: 0

Figlet:

Turns text into ascii art. Warning: too much text will break it
Args: 1 or more

Text:

Displays text thingys Args: 1 - (communism, unoreverse)
